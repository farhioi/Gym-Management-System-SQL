# SQL-Database-Design-Gym

## Project Overview

This is a course project focusing on the **design and implementation of a relational database** (SQL Server) for a hypothetical gym management system (members, classes, bookings).

## Key Skills Demonstrated

* **Database Design:** Modeled the entire system using an **ER Diagram**.
* **SQL Automation:** Used **Stored Procedures** (SPs) to automate data checks and improve workflow consistency.
* **Administration:** Implemented **Role-Based Access Control (RBAC)** for basic security management.
* **Documentation:** Created **Use Cases** to define system requirements.

## Repository Contents

* **`ER_Diagram.pdf`**: The visual Entity-Relationship Model.
* **`create_tables.sql`**: Script to build the full database schema.
* **`stored_procedures.sql`**: Scripts for automation and RBAC.
* **`Use_Cases_1-10.pdf`**: User requirements documentation.

## Setup

The project runs on **SQL Server**. Simply execute the `create_tables.sql` script, followed by `stored_procedures.sql`, in your SQL management tool to set up the environment.
